Небольшое веб-приложение, позволяющее создавать (и удалять) карточки (картинка и соответствующее ей слово)
и проходить тестирование по этим карточкам (показывается картинка, необходимо вписать слово);
просматривать свою статистику (таблица правильных и введенных ответов), а также общаться с другими
пользователями с помощью новостей, которые можно добавлять, редактировать, скрывать и удалять.

+ Можно выгружать новости через API в формате .json

Использованные модули и библиотеки: 
html/css/bootstrap
random
PIL
flask (+ flask-login, flask_wtf, ...)
wtforms
werkzeug
requests
sqlalchemy
datetime

Программа, запускающая проект - app.py
requirements.txt - файл с модулями и библиотеками, которые необходимо скачать
tz.txt - как readme, но tz
add_[...].py - файлы для ручного добавления объектов в базу данных

/templates - папка с HTML-файлами
/static/css/style/css - стилевой файл для HTML
/static/img/cards - папка с изображениями для общих карточек
/static/img/users_cards - папка с изображениями для личных карточек
/forms - файлы для формы регистрации, входа и изменения карточек и новостей
/db/ege_db.db - база данных
/data - все остальное: все файлы для БД + файлы для выгрузки новостей в API
